
Data Source:
The dummy data for this milestone was manually created for demonstration purposes.

Files and Their Contents:
- teams.csv: Contains team_id, team name, stadium, and location.
- players.csv: Contains player_id, name, age, weight, height, team_id, and position.
- player_stats.csv: Basic stats such as points, rebounds, assists, etc.
- team_stats.csv: Win/loss record and percentages.
- player_advanced_stats.csv: Advanced player metrics like true shooting % and PER.
